¿Qué es html?


"Standard Markup Language"
Describe la estructura de una paguina web
!DOCTYPE html : Define que el docuemnto es HTML5
html : El elemento raíz de una página html
head: Contiene meta información sobre la página
title: Especifica el titulo pata la página. Se muestra en la pestaña de la página
body: Define el cuerpo del documento. Contien todos lo documentos visibles.
h1: Define un encabezado grande
p: define un parrafo

Elementos de HTML

Un elemento es definido por un tag(Etiqueta), contenido o un tag final

NOTA: Alguno elemento no tiene contenido como "br". Esos Elementos llamados "Elementos vacios". No tiene etiqueta final.
